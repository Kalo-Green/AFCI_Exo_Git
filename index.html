Exercice Git

  • Se mettre dans le répertoire parent : cd chemin du répertoire
  • Création du répertoire : mkdir AFCI_Exo_Git
  • Création du fichier : touch index.html readme.md
  • Iniatialisation d'un référentiel Git : git init
  • Ajouter des fichiers à l’index pour être inclus dans le prochain commit : git add .
  • Création d'un commit : git commit -m "commit de base"
  • Création d'un nouveau dépôt sur GiHu
  • Indiquer la branche courante : git branch -M main
  • Création du dépôt distant : git remote add origin https://github.com/Kalo-Green/AFCI_Exo_Git.git
  • Pousser le commit vers le dépôt distant : git push -u origin main
